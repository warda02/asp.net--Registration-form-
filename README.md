# 🔐 ASP.NET Registration & Login System

A complete user authentication system built with ASP.NET and SQL Server — featuring signup, login, and credential verification.

## ✨ Features
- 📝 User registration with detail capture
- 🔑 Login with SQL Server credential check
- 🛡️ Secure authentication flow
- ✅ Input validation & error feedback
